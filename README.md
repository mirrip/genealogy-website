# Genealogy Website - Родословные Книги

Полноценный интернет-магазин родословных книг.

## Запуск

Просто откройте `index.html` в браузере:

```bash
# Linux
xdg-open index.html

# Mac
open index.html

# Windows
start index.html
```

Или перетащите файл в браузер.

## Функции

- Каталог книг с фильтрами
- Избранное
- Корзина покупок
- История заказов  
- Профиль пользователя
- Оформление заказа
- Адаптивный дизайн

## Товары

- Элитные издания в коже
- Изысканные книги
- Художественные издания
- Семейные альбомы
- Подарочные пакеты
