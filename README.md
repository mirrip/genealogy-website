# Genealogy Website

Современный SEO-first сайт для генеалогических услуг на Next.js 14.

## Запуск

```bash
npm install
npm run dev
```

Открой http://localhost:3000

## Структура

```
/app          - страницы (blog, services, cases, about, contact)
/components  - компоненты
/lib         - SEO утилиты
/styles      - стили
/public      - статические файлы
```
